## 精妙整理

使精妙系列自带的容器整理功能可以作用于玩家物品栏和所有可用容器。

### 主要功能

本模组为所有容器GUI添加了精妙核心中的两个整理按钮，它们会触发精妙核心的整理功能。

有四种整理模式：按名称、按模组、按数量、按标签，默认整理方式是按名称，按钮之一可以切换整理模式。

默认开启容器筛选，只有槽位数大于46的容器(物品栏36+容器10)会整理容器本身，其他情况下则会整理物品栏。

如，生存模式物品栏和工作台的槽位数都是46，所以会整理物品栏，而木桶、箱子等容器则会整理容器本身而不整理物品栏。

在无法整理容器自身的情况下，整理按钮不会显示，此时仍可通过快捷键进行物品栏整理，默认为R。

快捷键可在任何容器中生效，但触发容器整理的方式与按钮相同。

如果需要整理10个容器槽位及以下的容器，可以在配置文件中将容器筛选关闭。

本模组只是将精妙核心的整理功能拓展到所有容器，自身并没有整理功能，未来也不会加入更多与精妙核心无关的功能。

精妙系列的容器只能使用精妙核心自带的整理，否则堆叠升级将会失效，这也是本模组开发的原因。

对于习惯使用精妙系列的玩家来说，现在可以整个游戏统一使用一种整理了！
